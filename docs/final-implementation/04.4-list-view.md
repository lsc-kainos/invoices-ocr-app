# R4: Listagem e Visualização

> Requisito: "Allow users to see a list of all their previously uploaded documents along with the extracted information and LLM interactions."

---

## Visão Geral

O usuário pode ver todos os documentos que enviou, acompanhar o status de processamento, visualizar os detalhes extraídos e conversar sobre cada documento.

**Telas:**

- `/documents` — Listagem com filtros e resumo
- `/documents/[id]` — Detalhe com visualizador, campos extraídos e chat

## 1. Listagem de Documentos

### 1.1 SSR com Polling

```tsx
// apps/web/app/(authed)/documents/page.tsx
export default async function DocumentsIndex() {
  const cookieStore = await cookies();
  const cookieHeader = cookieStore
    .getAll()
    .map((c) => `${c.name}=${c.value}`)
    .join('; ');
  const res = await fetch(`${env.NEXTAUTH_URL}/api/documents?limit=100`, {
    headers: { cookie: cookieHeader },
    cache: 'no-store',
  });
  const docs = res.ok ? await res.json() : [];
  if (docs.length === 0) return <EmptyListState />;
  return <DocumentsList initialDocs={docs} />;
}
```

**SSR:** a listagem inicial vem do servidor. O componente `DocumentsList` usa `useDocumentsList` para fazer polling e atualizar status.

### 1.2 Componente DocumentsList

```tsx
// apps/web/components/features/documents-list/documents-list.tsx
export function DocumentsList({ initialDocs }: { initialDocs: DocumentSummary[] }) {
  const { docs } = useDocumentsList(initialDocs);

  return (
    <div className="flex h-full flex-col">
      <header>
        <h1>{t('title')}</h1>
        <span>{docs.length}</span>
      </header>
      <DocumentsSummaryCard docs={docs} />
      <div>
        {docs.map((d, idx) => (
          <DocumentRow key={d.id} doc={d} />
        ))}
      </div>
    </div>
  );
}
```

## 3. Backend: Endpoints

### 3.1 Listar Documentos

```typescript
// apps/api/src/documents/documents.controller.ts
@Get()
@Throttle({ default: { ttl: 60_000, limit: 1200 } })
list(
  @CurrentUser() user: User,
  @Query() query: ListDocumentsQueryDto,
): Promise<DocumentSummaryDto[]> {
  return this.docs.list(user.id, query);
}
```

**Parâmetros:** `limit`, `status`, `orderBy`

### 3.2 Detalhe do Documento

```typescript
@Get(':id')
findOne(
  @CurrentUser() user: User,
```

```typescript

@Patch(':id/summary')
updateSummary(
  @CurrentUser() user: User,
  @Param('id') id: string,
  @Body() body: UpdateDocumentSummaryDto,
```
